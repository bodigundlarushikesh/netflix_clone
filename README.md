# 🎬 StreamFlix - Netflix Clone

A Netflix-inspired movie streaming web application built using **HTML5, Tailwind CSS, JavaScript, and the TMDB API**. StreamFlix allows users to browse trending movies, search for their favorite titles, watch trailers, manage a personal watchlist, and enjoy a modern streaming experience through a responsive interface.

## 🌐 Live Demo

🔗 *https://netflix-clone-rho-tawny-94.vercel.app/**

## ✨ Features

- 🎥 Browse Trending Movies
- ⭐ Popular Movies
- 🏆 Top Rated Movies
- 📺 Popular TV Shows
- 🔍 Real-Time Movie Search
- 🎬 Watch Official Movie Trailers
- ❤️ Add & Remove Movies from Watchlist
- 👤 User Login & Signup

## 🔑 TMDB API

This project uses **The Movie Database (TMDB) API**.

Create your free API key:

https://developer.themoviedb.org/

Replace your API key inside:

```javascript
js/config.js
```

```javascript
TMDB_API_KEY: "YOUR_API_KEY"
```
